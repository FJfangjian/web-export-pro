## Chrome V3插件：网页表格导出Excel

从头创建了一个完整的Chrome Manifest V3插件。

### 项目结构
```
plugins-html-to-excel/
├── manifest.json              # V3 manifest
├── popup/                     # 弹出窗口
│   ├── popup.html
│   ├── popup.js
│   └── popup.css
├── content/                   # 内容脚本
│   ├── content.js             # 表格捕获、提取、导出核心
│   └── content.css            # 高亮样式、面板、Toast
├── background/
│   └── service-worker.js      # 后台服务
├── libs/
│   └── xlsx.full.min.js       # SheetJS 0.20.3
└── icons/                     # 自动生成PNG图标
```

### 核心功能
1. **捕获模式**：点击popup的"进入捕获模式"后，页面所有table被虚线高亮+行列数徽章标记
2. **单表导出**：点击任意table弹出预览面板 → 确认导出.xlsx
3. **全页导出**：一键导出所有表格，每个table一个sheet
4. **ESC退出**：捕获模式下按ESC退出

### 技术要点
- 使用SheetJS (xlsx.full.min.js) 进行Excel生成
- Content script注入所有页面，通过chrome.tabs.sendMessage通信
- CSS outline方式高亮table，不破坏页面DOM结构
- 合并单元格、嵌套表格等边界情况已处理
- 图标用Python+Pillow生成（渐变色+表格线+箭头符号）
